Github & Hugo搭建博客

创建的站点文件目录说明

创建的sites就是用来存放创建的网站的，创建的文件夹名为blog的网站：

|- archetypes ：存放default.md，头文件格式

|- content ：content目录存放博客文章（.markdown/.md文件）

|- data ：存放自定义模版，导入的toml文件（或json，yaml）

|- layouts ：layouts目录存放的是网站的模板文件

|- static ：static目录存放js/css/img等静态资源

|- config.toml ：config.toml是网站的配置文件

当前网站是没有任何内容的，需要下载个主题跑起来才有内容。

安装主题

|- archetypes ：存放default.md，头文件格式

|- layouts ：主题模板文件

|- static ：静态资源

|- theme.toml ：主题配置文件

运行Hugo server

hugo server –theme=mainraod hugo new post/test.md 注意需要将draft的值改为false才能看到页面

生成静态网站

hugo –theme=THEME_NAME baseUrl="http://YOURNAME.github.io/”

所有的静态页面都会生成道public目录下，生成静态网站后并push到你的github pages上。就能得到一个在线的个人博客了。

在github端创建repository

在命令行里边创建一个github repository

echo "# lknownothing.github.io" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin git@github.com:lknownothing/lknownothing.github.io.git
git push -u origin master

push an existing repository from the command line

git remote add origin git@github.com:lknownothing/lknownothing.github.io.git
git push -u origin master

在git bash里边检查ssh，并将ssh加入到ssh-agent，将.pub复制到github中。

  1. 执行指令 git add -A 将所有文件添加到仓库
  2. 执行指令：git commit -m “提交文件”
  3. 关联github仓库
    1. 到github仓库复制仓库地址。
    2. 执行指令：git remote add origin https://github.com/hanyuntao/text.git
    3. 上传本地代码：git push -u origin master

所遇到的问题

第一问题：

fatal: remote origin already exists.的解决方法。（报错远程源已经存在）解决方法

先输入git remote rm origin
再输入git remote add origin ***

第二个问题：

the authenticity of host “github.io” can’t be established. (建立主机与github.io的验证)

Appendix

1、Content文件头的设计格式

# Common-Defined params
title: "Example article title"
date: "2017-08-21"
description: "Example article description"
categories:
  - "Category 1"
  - "Category 2"
tags:
  - "Test"
  - "Another test"
menu: main # Optional, add page to a menu. Options: main, side, footer

# Theme-Defined params
thumbnail: "img/placeholder.jpg" # Thumbnail image
lead: "Example lead - highlighted near the title" # Lead text
comments: false # Enable Disqus comments for specific page
authorbox: true # Enable authorbox for specific page
pager: true # Enable pager navigation (prev/next) for specific page
toc: true # Enable Table of Contents for specific page
mathjax: true # Enable MathJax for specific page
sidebar: "right" # Enable sidebar (on the right side) per page
widgets: # Enable sidebar widgets in given order per page
  - "search"
  - "recent"
  - "taglist"
