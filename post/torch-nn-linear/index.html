Torch.Linear详解

August 09, 2020
强化学习, 教程

线性变化

线性变换公式

$$y=x A^{T}+b$$

我们将要输入的维度是(128,20) 想要输出 20维的特征

  1. dim=1
import torch

x = torch.randn(128, 20)  # 输入的维度是（128，20）
m = torch.nn.Linear(20, 30)  # 20,30是指维度
output = m(x)
print('m.weight.shape:\n ', m.weight.shape)
print('m.bias.shape:\n', m.bias.shape)
print('output.shape:\n', output.shape)

# ans = torch.mm(input,torch.t(m.weight))+m.bias 等价于下面的
ans = torch.mm(x, m.weight.t()) + m.bias   
print('ans.shape:\n', ans.shape)

print(torch.equal(ans, output))

验证输出

m.weight.shape:
  torch.Size([30, 20])
m.bias.shape:
 torch.Size([30])
output.shape:
 torch.Size([128, 30])
ans.shape:
 torch.Size([128, 30])
True

为什么weight的shape是（30,20), 实际运算中有转置。

matplotlib中ion()和inoff()的区别

其中的区别是

在交互模式下：

  • plt.plot(x)或者plt.imshow()直接出图像，不需要plt.show()
  • 脚本中使用ion()命令开启了交互模式，没有使用ioff的话，则图像会一闪而过。防止这种情况，需要在plt.show()之前加上ioff()命令。

在阻塞模式下：

  • 一个窗口关闭才能打开下一个新的窗口
  • plt.plot()或者plt.imshow()是直接出图像，需要plt.show()
