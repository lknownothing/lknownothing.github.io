OpenAI gym介绍

August 08, 2020
强化学习, 教程, Gym

介绍

目前，OpenAI Gym（以下简称gym）作为一个在强化学习领域内非常流行的测试框架，已然成为了Benchmark。

Toy example

import gym
test_envs={'algorithm':'Copy-v0',
           'toy_text':'FrozenLake-v0',
           'control':'CartPole-v0', # option：MountainCar-v0
           'atari':'SpaceInvaders-v0',# options：'Breakout-ram-v4'，'Seaquest-v0',
          'mujoco':'Humanoid-v1',     # not feasible on Win10
          'box2d':'LunarLander-v2' }  # not feasible on Win10

game_name = test_envs['algorithm']
env = gym.make(game_name)
for i_episode in range(20):
    observation = env.reset()
    for t in range(100):
        env.render() 
        print(observation)
        action = env.action_space.sample()
        observation, reward, done, info = env.step(action)
        if done:
            print("Episode finished after {} timesteps".format(t+1))
            break

在增强学习算法中有观察（observation）、回报（reward）、价值（value）、策略（policy）四个核心元素。使用python可以安装并加载OpenAI Gym提供的预设环境。这些环境在每一轮动作执行后以新的观察和即时回报作为输出，接受下一轮执行的动作作为输入。

Monte Carlo算法：与动态规划DP不同的是，这里不需要完整的知识，MC方法仅仅需要经验就可以求解最优策略。这些经验可以在线获得，或者根据某种模拟机制获得。MC方法是episode task上的，不管采取哪种策略。都会在有限时间达到终止状态并获得returnr任务。比如棋牌类游戏。有很多组这样经历完整的状态序列，就可以来近似估计状态价值了。进而求解预测和控制问题。

Model-free的强化学习：

  • 预测问题：即给定强化学习的5个要素：状态集S, 动作集A, 即时奖励R，衰减因子γ, 给定策略π， 求解该策略的状态价值函数v(π);
  • 控制问题，也就是求解最优的价值函数和策略。给定强化学习的5个要素：状态集S, 动作集A, 即时奖励R，衰减因子γ, 探索率ϵ, 求解最优的动作价值函数q∗和最优策略π∗.

MC优化考虑

  1. 同一状态可能在一个完整的状态序列中重复出现。比如，象棋中马可能重复到一个位置。两种方法：first visit和every visit。

  2. 累进更新平均值（incement mean）。average公式，意味着报错所有该状态的收获值，最优平均。浪费存储空间。一种方法就是保存上轮得到的收货均值与次数。

MC方法控制问题的算法流程

  1. 初始化所有的动作价值$Q(s,a)=0$,状态次数$N(s,a)=0$，随机初始化一个策略$\pi$

  2. k=k+1 基于策略进行k次MC采样，得到完整的状态序列。 $$ S_1,A_1,R_2,S_2,A_2,…S_t,A_t,R_{t+1},…R_T, S_T $$

  3. 对于每一个状态行为，计算收获$G_t$

  4. 基于新计算的动作价值，更新当前的贪婪策略

  5. 如果所有的Q收敛，则对应的Q(s,a)即为最优的动作价值函数，对应的策略$\pi(a|s)$为最优策略。

算法代码

算法完全遵循强化学习MC方法，rets为episode的累计奖励。

import numpy as np
import random
env=gym.make("Copy-v0")

def action_to_index(action):
    return action[0]*10+action[1]*5+action[2] # 0+5+4

def index_to_action(index):
    action=[index//10,index%2,index%5]
    return tuple(action)

STATE_COUNT=6  # ABCDE+边界
ACTION_COUNT=20  # 2*2*5 总共有20个动作

def init_MC(state_count,action_count):
    # 初始q值都为0
    q=np.random.rand(state_count,action_count)  
    # return G_t 一个动作之后的所有收货，计算V(s)
    rets=np.zeros((state_count,action_count),dtype=np.double)
    # 6个的策略
    policy=[random.randint(0,action_count) for _ in range(state_count)]
    return q,rets,policy
    
def learning(env):
    q,rets,policy=init_MC(STATE_COUNT,ACTION_COUNT)
    gamma=0.9
    epsilon=0.1
    total_score=0.0
    i_episode=0
    
    for i_episode in range(100000):
        total_reward=0
        observation=env.reset()
        g = np.zeros((STATE_COUNT, ACTION_COUNT), dtype=np.double)
        passed = np.zeros((STATE_COUNT, ACTION_COUNT), dtype=np.double)
        
        for t in range(10):
#             env.render()
            raw_action=policy[observation]
        
            if random.random()< epsilon:
                raw_action=action_to_index((random.randint(0,1),
                                           random.randint(0,1),
                                           random.randint(0,4)))              
            if passed[observation,raw_action]==0.0:
                passed[observation,raw_action]=1.0 # 记录visit
            action=index_to_action(raw_action)
            observation,reward,done,info=env.step(action)
           
            reward-=0.5 # 惩罚奖励为0的无用动作,即不write。
            total_reward+=reward
            
            # every visit
            for i in range(STATE_COUNT):
                for j in range(ACTION_COUNT):
                    passed[i][j]*=gamma
                    g[i][j]+=reward*passed[i][j]
            if done:
                break
                
            # 减小exploration chance
        if i_episode % 999==0:
                epsilon*=0.9
                
        rets+=g
        q=rets/(i_episode+1) # 每一个episode之后更新Q值
        policy=np.argmax(q,axis=1).tolist()

        total_score+=total_reward
    
    return policy,i_episode

def test_policy(env,policy):
    total_reward=0.0
    observation=env.reset()
    for t in range(1000):
        action=index_to_action(policy[observation])
        observation,reward,done,info=env.step(action)
        total_reward+=reward
        if done:
            break
    print("Total reward: %f"% total_reward)

# env.monitor.start("Copyv0-experiment-0")
policy,n_episode=learning(env)
# env.monitor.close()

print("Final policy:"+str(policy))
print("Episode trained:"+str(n_episode))
test_policy(env,policy)
